Arrays in javascript.

The join() method in JavaScript is a way to take all the items in an array and turn them into a single string. If you want, you can put a specific character, like a comma or a dash, between each item in the string. Normally, if you don't choose a character, the method will use a comma

In summary, slice() is used to extract parts of an array without modifying it, whereas splice() is used to change the content of an array by removing, replacing, or adding elements
